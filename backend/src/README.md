# backend/src 目录说明

## 文件夹功能

保存后端正式运行代码。

## 文件夹内容

- `api/`: HTTP 与 WebSocket 接口。
- `application/`: 运行时装配和跨模块用例编排。
- `database/`: 数据库连接、schema 和迁移。
- `dialogue/`: 对话生成与上下文构造。
- `memory/`: 图记忆、向量检索与记忆维护。
- `npc/`: NPC 状态、任务计划、affordance 校验、节点追踪和社交决策。
- `save/`: 存档读写与恢复。
- `world/`: 时间、地点、事件和世界状态。
- `prompting/`: Prompt task 注册、标签兼容和 messages 组装。
