## 待用户操作（Unity Editor 手动）

1. **创建 NPC 预制体** `Assets/Prefabs/NpcPrefab.prefab`：
   - 根节点挂 `NpcEntity.cs`
   - 子节点：Sprite (SpriteRenderer) + NameLabel (TextMeshPro) + BubbleCanvas (Canvas, World Space)
   - 把子节点引用拖到 NpcEntity 的 SerializeField 槽位

2. **场景挂载**：
   - 创建空 GameObject → 挂 `NpcSpawner.cs` → 拖入预制体
   - 创建空 GameObject → 挂 `NpcBubbleManager.cs` → 拖入 NpcSpawner 引用

3. **端到端测试**：
   - 启动后端 `cd backend && python run.py`
   - Unity Play → 观察 NPC 生成/移动/对话气泡
