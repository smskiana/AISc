## 实际改动

1. 移除玩家正式对话首轮流式生成的 `max_tokens=120`。
2. 移除玩家选择后的后续轮流式生成的 `max_tokens=120`。
3. 保留现有 Prompt 中“回答一般不超过 80 字（除非在讲故事或对方追问）”的软长度约束。
4. 摘要、回复建议、记忆路由和 NPC 行为等需要结构化输出或独立预算的调用不在本次范围内。

## 诊断钩子检查
