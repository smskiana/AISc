# 世界存档权威迁移设计方案

> 执行记录: [execution.md](2026-07-14_世界存档权威迁移_execution.md)

## 需求理解

完成现有持久化入口迁移：Unity 本地 manifest 提供存档列表和读档身份，Python 完整存档退出默认链路；Python 世界表只作为运行时推理缓存和旧协议兼容。

## Workstream / Roadmap

- Workstream: `ProtocolAndSave`
- Roadmap item: 旧后端世界状态迁移
- 相关 ADR: ADR-0006

## 实现方案

1. Unity 仓储提供本地存档列表。
2. 开始菜单通过 `GameManager` facade 读取 Unity 列表。
3. 读档完成直接形成 `GAME_READY` 兼容事件，不再调用 Python `CONTINUE` 覆盖状态。
4. 玩家、时间和 NPC 世界事实持久化只来自 Unity `GameSaveData`。
5. Python `SaveManager` 与旧命令保留兼容标记，但不再被 Unity 默认调用。

## 验证

运行 Python 全套测试、Unity MCP 编译 / Console、项目规范检查和关键符号静态检查。
