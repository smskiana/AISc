# 重连与世界快照设计方案

> 执行记录: [execution.md](2026-07-14_重连与世界快照_execution.md)

## 需求理解

重连和读档后由 Unity 发送全量世界事实快照，覆盖 Python 推理缓存；临时对话、任务跟踪和流式状态不恢复。

## Workstream / Roadmap

- Workstream: `ProtocolAndSave`
- Roadmap item: 重连握手与世界快照
- 相关 ADR: ADR-0006

## 实现方案

1. `hello_ack` 后 Unity 自动发送 `world_snapshot`。
2. 快照包含 revision、checkpoint、时间、玩家位置和 NPC 世界事实。
3. Python 在事务内更新兼容缓存表，并返回已应用 revision。
4. 重连时清理对话准备态、活跃态和未确认 NPC 任务追踪。
5. Python 缓存永远不反向覆盖 Unity。

## 验证

Python 测试覆盖快照更新与临时状态清理；Unity MCP 编译和 Console 检查。
