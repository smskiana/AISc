# docs 目录说明

## 文件夹功能

本目录保存项目设计、工程口径、架构决策、实施证据和问题记录。

## 文件夹内容

- `ProjectIndex.md`: 项目功能总入口。
- `Roadmap.md`: 长期目标和阶段安排。
- `Workstreams/`: 各功能域的当前工程口径。
- `DecisionRecords/`: 架构决策记录。
- `DesignDocs/`: 设计文档和工程规范。
- `AIChanges/`: 按功能分类的实际实现证据。
- `KnownIssues.md`: 未解决和暂缓问题。

查找项目内容时先进入 `ProjectIndex.md`，不要直接按日期浏览 `AIChanges`。
